Bug reports

If you think there is something really wrong with PM123. Please send a detailed bug report to the contact address. The bug report should include the following information:

  1. the exact version of PM123, e.g. 'PM123 1.40 alpha 3',
  2. a step by step description how to reproduce the problem,
  3. the plug-ins used and
  4. optionally a logfile from PM123. Please compress the log output and do not send mails with several megabytes attached, since they most likely won't arrive.
