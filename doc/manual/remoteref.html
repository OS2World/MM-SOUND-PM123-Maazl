PM123's Remote Command Reference

PM123 currently understands the following commands through the remote interface. The commands are grouped into five categories: playback control, playlists, user interface, meta data and configuration.

Many commands return common error codes. All commands renturn an empty line at the end. This is different from PM123 up to version 1.40. However, unless you send no more than one command per pipe instance it makes no difference.

Note that the commands must have an asterisk (*) as prefix if the are sent through the pipe.

Alphabetical list of all commands

  • add - add item to playlist
  • autouse - auto use playlist on add
  • cd - set current base URL or directory
  • clear - clear playlist
  • current - query current song or playlist
  • dir - add directory to playlist (deprecated, use add)
  • float - float window on top
  • font - select font
  • forward - forward scan
  • hide - hide player window
  • info format - query file and format information
  • info meta - query meta information
  • info pl_item - query information about the current playlist item
  • info playlist - query information about a (nested) playlist
  • jump - jump to location
  • load - load file/URL 
  • location - query current location
  • navigate - navigate to item
  • next - skip next
  • open - open dialog window
  • option - set/query configuration options
  • pause - pause
  • pl_current - query current playlist item
  • pl_index - query current playlist item index
  • pl_next - next playlist item
  • pl_prev - previous playlist item
  • pl_reset - reset playlist iterator
  • play - play item
  • playlist - select playlist
  • playonload - start playing on load
  • prev, previous - skip previous
  • query - query playback mode (like playing, auto repeat and so on)
  • quit - quit player
  • rdir - recursive add directory to playlist (deprecated, use add)
  • remove - remove item from playlist
  • repeat - autorepeat mode
  • rewind - reverse scan
  • save - save playlist
  • savestream - save stream as file
  • size - set player size
  • shuffle - enable shuffle
  • skin - load skin
  • status - get info string
  • stop - stop playback
  • stopat - stop playback at a certain location
  • use - use playlist
  • volume - volume control
