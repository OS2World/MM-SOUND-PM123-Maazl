Keyboard shortcuts

Key Main Window Playlist, Playlist Manager, Bookmarks
Enter Play/Stop. Navigate to the selected item.
Pause key Pause.  
Left arrow Rewind.  
Right Arrow Fast forward.  
Up arrow Previous song. Cursor up
Down arrow Next song. Cursor down
+/- Increase/decrease volume by 5%.  
Space Open main menu. Open the context menu for the selected item(s).
Escape Cancel slider drag.  
Ctrl+I Show info window of the currently loaded object (maybe a playlist). Show info window of the currently open playlist.
Ctrl+Shift+I Show info window of the current song. Show info window of the currently selected object (playlist or song).
Ctrl+E Edit meta information of the current song. Edit meta information of the currently selected song.
Ctrl+S Save stream as... Save playlist as...
Ctrl+O Load other file(s) playlist(s) or a folder in the player. Open another playlist (in the same view as the current).
Alt+U Load an URL or file directly by name. Append an URL to the playlist.
Alt+Shift+U Load an URL or file directly by name. Append an URL to the currently selected playlist.
Insert   Append file(s) playlist(s) or a folder to the playlist.
Shift+Insert   Append file(s) playlist(s) or a folder to the currently selected playlist.
Alt+T Open CD track(s). Append CD track(s) to the playlist.
Alt+Shift+T   Append CD track(s) to the currently selected playlist.
Delete   Remove the currently selected item(s) from the playlist.
Ctrl+C   Clear the playlist.
Ctrl+U   Use this playlist.
Ctrl+Shift+U   Load the currently selected item(s) into the player.
Ctrl+P Open default playlist in editor.  
Ctrl+M Open Playlist Manager.  
Ctrl+D Open the currently loaded playlist in the editor. Open the playlist in the editor.
Ctrl+Shift+D   Open the currently selected playlist in the editor.
Ctrl+T Open the tree view of currently loaded playlist. (Playlist Manager) Open the tree view of the playlist.
Ctrl+Shift+T   Open tree view of the currently selected playlist.
Ctrl+B Bookmark currently loaded object (maybe a playlist).  
Ctrl+Shift+B Bookmark current song.  
Ctrl+L Bookmark currently loaded object (maybe a playlist) including the current location.  
Ctrl+Shift+L Bookmark current song including the current playing location.  
Alt+B Open the Bookmark editor.  
Alt+Shift+B Open the Bookmark editor as tree view.  
Alt+Shift+I Show the Inspector Window.
