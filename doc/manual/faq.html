PM123 FAQ

How to organize my Bookmarks hierarchically?

With PM123 1.40 the bookmarks in the context menu are no longer flat. They may be organizes in sub menus like with a web browser.

  1. Open the Bookmarks window as tree view.
  2. Right click on the free space in the bookmarks area to open the context menu of the playlist.
  3. Select Append.../new Playlist...
  4. Select an appropriate file location for your new bookmark section. The file need not to exist.
  5. Use drag and drop to move bookmarks into this new folder.
Nested playlists will automatically show up as sub menus in the bookmark list.

How do I manage Internet Radio Stations?

The recommended way is to use a playlist. Put the URLs of the Internet Radio Stations into this playlist. While playing a stream of this playlist the previous/next buttons change the program.

Since an internet stream usually does not end the playlist is never played as sequence of songs. Instead the current stream keeps playing until you force another one to load.

How to deal with playlists of Internet Radio Stations with alternate server URLs?

  1. Put the playlist URL of the web site into your playlist with internet radio stations.
  2. Open the context menu of the new entry from your local playlist and chose Properties (Alt-Enter).
  3. Check the box Alternation list.
  4. Set either clear shuffle or force shuffle (see below).

This will take the whole playlist as one item. So using previous/next does not propagate to the next entry in this playlist, but leave the sub list and continue in the parent playlist.

Clear shuffle enforce PM123 to play the first valid item in the list of URLs although if the shuffle button is pressed in the main window.
Force shuffle causes PM123 always to take a random entry from the list of streams. This is effectively causes load balancing.
If in doubt chose clear shuffle.
