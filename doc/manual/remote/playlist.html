PM123's Remote command reference - Playlist commands

See also playlist manipulation commands.

Command Description Reply
playlist <url> (1.) Select a playlist or a folder. This setting only affects the pipe interface. To play a playlist use the load command. URL or empty line on error.
pl next [<count>] (2.) Navigate to the next item in the current playlist. If <count> is specified, you can skip more than one item at once. URL of the item or an empty line at the end of the playlist.
pl prev [<count>] (2.) Navigate to the previous item in the current playlist. If <count> is specified, you can skip more than one item at once. URL of the item or an empty line at the start of the playlist.
pl reset (2.) Reset the current item of the playlist. The current location is now before the start or past the end respectively.  
pl current Query the current playlist. URL of the item or an empty line if none.
pl item Query the current item in the current playlist. URL of the item or an empty line if none.
pl index Query the index of the current item within the current playlist.
Be careful, because the index changes when the playlist is modified and you might be not the only one that modifies a playlist.		 Index starting from 1 or empty in case of no current item.
use Use the current playlist. This is equivalent to 'load <URL to current playlist>'. Result code, non-zero = error.

Notes

  1. If you pass relative URLs the path set with the cd command is used as root. Note that this does not apply to location string.

  2. In PM123 before version 1.40 the playlist manipulation commands always operate on the currently playing item. Now the new commands pl prev, pl next and pl reset are dedicated to playlist operations.
