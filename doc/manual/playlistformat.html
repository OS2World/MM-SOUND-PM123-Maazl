Format of the PM123 playlist files

This format is used for PM123 playlists as well as for bookmarks, playlist manager and most recent use lists. The files are upwardly and downwardly compatible to the files of older PM123 releases, concerning the least common denominator.

Header

The head of a playlist file is a comment.

#
# Playlist created with PM123 1.40 alpha 4
# Do not modify!
# Lines starting with '>' are used by Playlist Manager.
#

For each song entry

  • Alias name for display, e.g. in the bookmark list. optional

    #ALIAS Recording from Mar. 03th, 2006

  • Information on start position. optional
    The unit is dd hh:mm:ss, seconds with fractionals.

    #START 3.921

  • Information on stop position. optional
    The unit is dd hh:mm:ss, seconds with fractionals.

    #STOP 3:04.122

  • Information on last playing position, e.g. in the bookmark list. optional
    The unit is dd hh:mm:ss, seconds with fractionals.

    #AT 3:02.135

  • Human readable comment with size, duration and an info string

    # 22385.124 kB, 21:18, 143.4kbps, 44.1kHz, Stereo

  • Filename or URL

    SL060319_2204.ogg

    On output PM123 writes Filenames with relative or absolute paths, and URLs. On input file:/// URLs are accepted too. In fact any URL which is accepted by PM123 is accepted. This may depend on the loaded plug-ins.

  • Cached information about the above entry.

    >143,44100,2.0,22923026,1278.7

    This is the bit rate [kbps] or -1 if unknown,
    the sampling rate [Hz],
    the number of channels as mode (0=stereo, 3=mono) or as channels.lfe-channels, e.g 2.0=stereo,
    the file size [bytes] or -1 if unknown and
    the song length [s] or -1 if unknown.

For each playlist or folder entry

  • Alias name for display, e.g. in the bookmark list. optional

    #ALIAS Chillout-1

  • Information on start position. optional
    The format is a PM123 location string.

    #START "SL060319_2224.ogg";3:58.322000

  • Information on stop position. optional
    The format is a PM123 location string.

    #STOP "SL060319_2354.ogg";2:21.122000

  • Human readable comment with file size, total duration and an info string

    # 1kiB, 1:15:18,

  • Filename or URL

    Taucher - Adult Music\Chill.lst

    On output PM123 writes Filenames with relative or absolute paths, and URLs. On input file:/// URLs are accepted too. Folder entries must end with a trailing / or \.

  • Technical information about the last entry

    >136,-1,-1,896,1278.7,145,19775.4,15,0

    • 136 - This is the average bit rate [kbps] or -1 if unknown,

    • -1,-1 - two constants indicating unknown sampling rate and channels,

    • 896 - the file size [bytes] or -1 if unknown,

    • 1278.7 - the total length [s] or -1 if unknown,

    • 145 - the number of song items (including sub items) in the referenced playlist,

    • 19775.4 - the total file size of all song items (including sub items),

    • 15 - the number of items in the referenced playlist,

    • 0 - a flag whether the content is known to contain itself recursively is added,


Footer

# End of playlist

The footer is optional on input.
