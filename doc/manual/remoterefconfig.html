PM123's Remote command reference - Configuration control

Command Description Reply
version Return the PM123 core version. 1.41 alpha 4 (for example)
option [<plug-in>:]<name> [<new value>] Sets or queries an option value. If <new value> is missing only the current value is returned.
To access plug-in specific options precede the option with name of the plug-in and a colon. e.g. mp123: 		Previous setting or empty string on error.
autouse [on|off|toggle]1 Toggles or (re)sets 'auto use playlist on add'.
Without an argument only the current setting is returned.		 Previous setting or empty string on error.
playonload [on|off|toggle]1 Toggles or (re)sets 'auto play on load'.
Without an argument only the current setting is returned.		 Previous setting or empty string on error.
float [on|off|toggle]1 Toggles or (re)sets 'float on top'.
Without an argument only the current setting is returned.		 Previous setting or empty string on error.

Notes 

  1. All boolean parameters may use on, 1, true or yes for the logical value true and off, 0, false or no for false. The token toggle flips the current setting.
