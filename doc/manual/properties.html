PM123's Configuration Dialog


This properties dialog can be accessed from the main menu. In the dialog you can configure PM123's engine and interface. It consists of:
  • The configuration settings page 1 and page 2,
  • the display configuration,
  • the plug-in configuration for decoders, filters, output plug-ins and visual plug-ins.
  • last but not least the About dialog. There are some info about PM123, author etc.
