PM123's Command line Parameters

PM123 accepts a couple command line parameters:

-cmd [host] command
If `host' is omitted, send commands to local PM123, otherwise send commands to PM123 running on `host'. For a command listing, see PM123 Remote control.
-shuffle
Shuffles files added from command line.

PM123 also accepts MP3 files, URLs, playlists, directories and CD tracks as command line parameters. The CD track can be specified as cd:///J:/Track XX, where J is a CD drive letter and XX is a CD track number.
